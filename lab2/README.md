# Лабораторная работа №2: Стековый калькулятор

## 📝 Описание
Реализация стекового калькулятора с использованием шаблона проектирования «Фабричный метод», системы журналирования (logging) и модульного тестирования (unit testing).

## 🎯 Цель задания

### Часть 1 (15 баллов)
Разработать стековый калькулятор, который:
- Принимает имя файла с командами как аргумент командной строки
- При отсутствии аргумента использует стандартный поток ввода (stdin)
- Работает с вещественными числами

### Часть 2 (10 баллов)
1. Реализовать модульные тесты, покрывающие функционал калькулятора
2. Реализовать журналирование процесса работы

## 📋 Команды калькулятора

| Команда | Описание | Пример |
|---------|----------|--------|
| `#` | Комментарий (строка игнорируется) | `# Это комментарий` |
| `PUSH x` | Положить число x на стек | `PUSH 5` |
| `POP` | Снять число со стека | `POP` |
| `+` | Сложение двух верхних элементов | `+` |
| `-` | Вычитание двух верхних элементов | `-` |
| `*` | Умножение двух верхних элементов | `*` |
| `/` | Деление двух верхних элементов | `/` |
| `SQRT` | Квадратный корень из верхнего элемента | `SQRT` |
| `PRINT` | Печать верхнего элемента (без удаления) | `PRINT` |
| `DEFINE name value` | Задать значение параметра | `DEFINE a 4` |
