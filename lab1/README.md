# Лабораторная работа №1: Подсчет частоты слов

## 📝 Описание
Программа на Java для анализа текстовых файлов и генерации CSV-файла со статистикой частоты слов. Лабораторная работа демонстрирует использование контейнеров, потоков ввода-вывода и ключевых классов Java (Object, String).

## 🎯 Цель задания
Написать программу, которая:
- Принимает имя текстового файла как аргумент командной строки
- Обрабатывает файл для подсчета частоты слов
- Генерирует CSV-файл со следующими колонками:
  1. **Слово** - найденное слово в тексте
  2. **Частота** - количество вхождений слова
  3. **Частота (в %)** - процент относительно общего количества слов
- Результат сортируется по убыванию частоты (самые частые слова в начале)
- Разделителями считаются все символы кроме букв и цифр

## 📋 Требования

### Технические требования
- Использовать `java.lang.StringBuilder` для построения слов
- Использовать `java.io.InputStreamReader` и `FileInputStream` для чтения файлов
- Использовать `Character.isLetterOrDigit()` для определения класса символа
- Хранить статистику в памяти используя реализацию интерфейса `java.util.Set`
- Создать специальный класс для хранения пар слово-счетчик
- Реализовать методы `equals()` и `hashCode()` для использования HashSet

### Ввод/Вывод
- **Входные данные:** Имя текстового файла (аргумент командной строки)
- **Выходные данные:** CSV-файл со статистикой слов
